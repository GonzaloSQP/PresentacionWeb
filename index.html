SKILLS

C#

C# es un lenguaje de programación orientado a objetos desarrollado por Microsoft como parte de su plataforma .NET. Se utiliza para desarrollar aplicaciones de escritorio y de servidor en Windows, así como aplicaciones móviles y juegos en la plataforma Unity.

Java

Java es un lenguaje de programación orientado a objetos y una plataforma informática que permite ejecutar aplicaciones en múltiples dispositivos y sistemas operativos. Se utiliza ampliamente para desarrollar aplicaciones web, móviles y de escritorio.

JavaScript

JavaScript es un lenguaje de programación interpretado y orientado a objetos que se utiliza principalmente para crear efectos interactivos en páginas web y aplicaciones web. Es compatible con todos los navegadores web modernos y se puede ejecutar tanto en el lado del cliente como en el lado del servidor.

CSS / Bootstrap

CSS es un lenguaje de diseño utilizado para definir el estilo visual de páginas web y aplicaciones. Permite separar la estructura y contenido de un documento HTML de su presentación visual.

.Net

.NET es una plataforma de desarrollo de software desarrollada por Microsoft que proporciona una gran cantidad de herramientas y servicios para crear aplicaciones de escritorio, web y móviles en diferentes lenguajes de programación, como C#, VB.NET y F#.

AWS - Windows Server

AWS (Amazon Web Services) es una plataforma de servicios en la nube ofrecidos por Amazon que proporciona servicios de almacenamiento, procesamiento, análisis y seguridad en línea. Es ampliamente utilizado por empresas y desarrolladores para alojar aplicaciones y sitios web en la nube.

Mysql / Sql Server

SQL (Structured Query Language) es un lenguaje de programación utilizado para administrar y manipular bases de datos relacionales. Se utiliza para realizar consultas, actualizaciones y modificaciones de datos en la base de datos.

AWS S3

AWS S3 (Amazon Simple Storage Service) es un servicio de almacenamiento de objetos en la nube ofrecido por Amazon Web Services. Permite almacenar y recuperar datos desde cualquier lugar de Internet y se utiliza ampliamente para almacenar y distribuir contenido estático, como imágenes, videos y archivos.

AWS - EC2

AWS EC2 (Amazon Elastic Compute Cloud) es un servicio de infraestructura de nube que permite a los usuarios alquilar y escalar servidores virtuales en la nube. Es utilizado por empresas y desarrolladores para alojar aplicaciones y sitios web, procesamiento de datos y análisis, y para ejecutar software especializado en la nube.

AWS - LAMBDA

AWS Lambda es un servicio de computación sin servidor ofrecido por Amazon Web Services que permite a los usuarios ejecutar código sin tener que administrar servidores. Es utilizado por empresas y desarrolladores para ejecutar código en respuesta a eventos, como solicitudes de API, cargas de archivos y eventos de base de datos.

GIT - GIT HUB

Git es un sistema de control de versiones utilizado para el seguimiento de cambios en archivos de código fuente y otros recursos informáticos. GitHub es una plataforma en línea para alojar, compartir y colaborar en proyectos de software utilizando Git.
